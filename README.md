# App-Store
An App Store allowing users to download, update, and use my personal projects, downloadable on my site.
The source code is private for security reasons.

### Tools used  
Frontend - Electron  
Backend - AWS (API Gateway, Lambda, DynamoDB and S3)  

### User Data Storage  
ID of the user (No method to identify the ID owner)  
For each app the user has installed, which version they currently have
